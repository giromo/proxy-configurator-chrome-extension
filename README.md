### Manual Installation

1. **Download the ZIP**:
   - Click [here](https://github.com/yourusername/proxy-configurator-chrome-extension/raw/main/proxy-configurator-chrome-extension.zip) to download the extension.
2. **Unzip the File**:
   - Extract the ZIP file to a folder (e.g., `proxy-configurator-chrome-extension`).
3. **Load in Chrome**:
   - Open Chrome and go to `chrome://extensions/`.
   - Enable **Developer mode** (top-right toggle).
   - Click **Load unpacked** and select the extracted folder.
4. **Use the Extension**:
   - Click the extension icon in your toolbar to configure your proxy settings.

## üöÄ Quick Start

### Basic Configuration
1. **Host**: Enter your proxy server address (e.g., `proxy.example.com` or `192.168.1.100`)
2. **Port**: Enter the proxy port (e.g., `1080` for SOCKS5)
3. **Protocol**: Choose SOCKS4 or SOCKS5
4. **Authentication** (optional): Enter username and password if required
5. **Connect**: Click the Connect button
